
# CafePay Wallet 💳

Smart student spending with intelligent meal plan and expense management. Built with Next.js, Firebase, and the **CafePay AI Engine** (Decision Tree Analysis).

## Project Title
**Design and Development of a Web-Based Intelligent Expense Tracking System Using Purchase-Based Logging and Decision Tree Analysis.**

## Core Technologies
- **Next.js 15**: High-performance App Router for seamless navigation.
- **Firebase**: Secure real-time data persistence and authentication.
- **Genkit AI**: Intelligent spending classification and decision tree pathing.
- **PWA Ready**: Installable on Android and iOS devices for campus use.

## Academic Alignment
- **Purchase-Based Logging (PBL)**: Automatic synchronization between vendor orders and expense records.
- **Decision Tree Analysis**: Algorithmic classification of spending behavior into Tactical status categories.
- **Financial Leakage Detection**: Identifying high-frequency low-value "leakage" points using frequency nodes.

## Getting Started
To get started with the project, take a look at `src/app/page.tsx`.

## Project Structure
- `/src/app`: Next.js App Router pages.
- `/src/firebase`: Firebase configuration and optimized hooks.
- `/src/ai`: Genkit AI flows for the CafePay AI Engine.
- `/src/components`: Reusable glassmorphism UI components.
